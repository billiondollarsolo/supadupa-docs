# Compliance And Audit

Supadupa includes compliance, advisor, access review, and audit surfaces to help operators gather evidence.

:::warning
These screens are operator evidence helpers. They are not a certification claim.
:::

## What To Use Them For

- Review audit events.
- Inspect access posture.
- Track security advisor findings.
- Collect operational evidence during internal review.

## What They Do Not Prove Alone

- Formal compliance certification.
- Complete infrastructure security.
- Backup restore success.
- External provider propagation.
